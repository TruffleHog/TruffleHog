\subsection{Detailed Description}
\hyperlink{classedu_1_1kit_1_1trufflehog_1_1view_1_1_abstract_network_graph_view}{Abstract\+Network\+Graph\+View} is the basic abstraction for every implementation of a network graph view. By extending the Swing wrapper class \hyperlink{}{Swing\+Node} its possible to place an instance of a \hyperlink{classedu_1_1kit_1_1trufflehog_1_1view_1_1_abstract_network_graph_view}{Abstract\+Network\+Graph\+View} into the existing javafx environment easily.
