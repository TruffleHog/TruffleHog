\subsection{Detailed Description}
A \hyperlink{classedu_1_1kit_1_1trufflehog_1_1service_1_1replaylogging_1_1_replay_log}{Replay\+Log} object contains the snapshot of a given graph, a list of all commands that were executed for the next X seconds following that snapshot and a timestamp of when the \hyperlink{classedu_1_1kit_1_1trufflehog_1_1service_1_1replaylogging_1_1_replay_log}{Replay\+Log} starts to where it ends. 

\hyperlink{classedu_1_1kit_1_1trufflehog_1_1service_1_1replaylogging_1_1_replay_log}{Replay\+Log} objects are used to recreate a graph that already occurred. They capture all necessary information to display the full graph the way it was at any given point in time. This is done by applying the commands stored in it back on the snapshot until the end of the \hyperlink{classedu_1_1kit_1_1trufflehog_1_1service_1_1replaylogging_1_1_replay_log}{Replay\+Log} is reached, at which point the commands of the next replay log are applied and so on.
